The <a href="https://cland.lsce.ipsl.fr/">CLAND</a> project involved my <a href="https://www.theses.fr/2022UPASB012">PhD work</a>, which aimed to quantify C inputs needed to reach a 4‰ target. First, we developed an <a href="https://bg.copernicus.org/articles/18/3981/2021/">inverse modeling framework</a> with the Century model to assess the amount of C input required to increase soil organic carbon stocks by 4‰ per year. Second, we applied this framework to a <a href="https://bsssjournals-onlinelibrary-wiley-com.insu.bib.cnrs.fr/doi/full/10.1111/ejss.13330">multi-model ensemble</a>, in order to assess the uncertainty linked to the model structure. Finally, we proposed a parametrization approach to upscale the simulations, and map the C input requirements at the Euoropean scale. Holisoils featured my PhD work in a <a href="https://holisoils.eu/is-it-feasible-to-increase-soil-carbon-stocks-by-4-per-1000-per-year/">blog post</a> summarizing some of the results.
